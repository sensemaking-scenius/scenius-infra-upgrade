# Scenius Infra Upgrade

Structured deliberation archive for the Sensemaking Scenius infrastructure upgrade project.

Sessions are synced automatically from [Harmonica](https://harmonica.chat) using [harmonica-sync](https://github.com/harmonicabot/harmonica-sync).

## Sync

```bash
export HARMONICA_API_KEY=hm_live_...
npx harmonica-sync
```
